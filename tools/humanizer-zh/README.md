# Humanizer-zh: AI 写作去痕工具（中文版）

本工具是一个专为 AI Agent（如 Claude Code 或 Antigravity）设计的文字编辑技能（Skill），通过内嵌维基百科 "Signs of AI writing" 的 24 种写作特征检测规则，帮助您识别并去除文本中生硬的 AI 痕迹，输出更自然、更具人类思考深度的中文文本。

---

## 🛠️ 第一阶段：环境自检与首次初始化引导

为了使 AI Agent 能够自动并顺畅地运行本技能，我们需要执行以下诊断与加载步骤。

### 1. 依赖与路径自检
本技能为纯文本提示词规则库（`SKILL.md`），无任何第三方运行语言（如 Python、Node.js）的代码级依赖。但为了让系统正确加载，需要验证其安装路径。
* **路径验证**：
  在您的 AI Agent 客户端中，确保本技能的 `SKILL.md` 存在于以下正确的技能发现目录下：
  * **Antigravity 全局路径**：`C:/Users/<您的用户名>/.gemini/config/skills/humanizer-zh/`
  * **Antigravity 项目局域路径**：项目根目录下的 `.agents/skills/humanizer-zh/`
  * **Claude Code 默认路径**：`~/.claude/skills/humanizer-zh/` (Windows 对应 `%USERPROFILE%/.claude/skills/humanizer-zh/`)

* **自检核对清单**：
  在目标文件夹中必须包含以下文件：
  * `SKILL.md` (包含前置元数据 YAML 和核心去痕规则)
  * `README.md` (本说明文档)

### 2. 交互配置与凭证自愈
* **API/凭证依赖**：本工具**不需要**任何外部 API 密钥、数据库凭证或网络 Cookie。它完全基于宿主 LLM 本身的推理能力运行。
* **首次加载验证**：
  在终端中启动您的 Agent 并输入以下指令以验证是否成功加载：
  ```
  /humanizer-zh
  ```
  若加载成功，Agent 将会确认此技能已激活，并进入去 AI 痕迹文字编辑状态。

## 🚀 第二阶段：核心执行工作流

本技能的工作流分为**自动运行**和**手动检测**两种模式。

### 1. 核心路由与激活方式
* **触发场景**：当您需要“降AI味”、“去AI腔”、“人性化润色”、“改写AI文案”或“审校AI生成的博客/长文”时，技能将被自动激活。
* **参数/命令语法**：
  * **指令式激活**：
    ```text
    /humanizer-zh 请帮我人性化以下文本：[您的文本]
    ```
  * **上下文式激活**：
    ```text
    请用 humanizer-zh 帮我改写这段话，让它更自然：[您的文本]
    ```
  * **文件处理**：
    ```text
    /humanizer-zh 请人性化 [article.md](file:///path/to/article.md) 中的内容
    ```

### 2. 核心诊断与改写流程
当技能激活后，Agent 会严格遵循以下 4 步流程处理文本：
1. **模式扫描**：对照 24 种典型的 AI 写作痕迹（如：过度强调意义“充当/作为”、肤浅分析的 -ing 短语、无中生有/模糊的归因“专家指出”、公式化结构等）对文本进行全量扫描。
2. **节奏与结构打破**：删除多余的连接词（“此外”、“值得注意的是”）、打破三段式排比、调整句子长度交错。
3. **注入人味与观点**：根据文本风格，在适合的地方引入人类视角，补充具体功能和真实细节，避免纯粹“无菌”的中立说明。
4. **质量评估打分**：自动依据 5 维评分表（直接性、节奏、信任度、真实性、精炼度）进行打分（总分 50 分），仅在得分 $\ge 35$（推荐 $\ge 45$）时进行最终输出。

### 3. 工具卸载
若您不需要此技能，可直接删除上述自检路径中的 `humanizer-zh` 文件夹，或使用以下命令清理：
* **Antigravity 中删除**：
  直接在全局或项目技能目录下物理删除 `humanizer-zh/` 文件夹。
* **Claude Code 中删除**：
  ```bash
  rm -rf ~/.claude/skills/humanizer-zh
  ```

## 📝 附录：检测的 24 种 AI 写作模式与高警示词

* **常见高频 AI 词汇**：此外、至关重要、深入探讨、格局（抽象名词）、织锦、持久的、充满活力的。
* **核心去痕原则**：
  1. 删除填充短语与谄媚前言。
  2. 句子两项优于三项（避免过度三段式）。
  3. 适当使用第一人称“我”，允许合理的题外话。
