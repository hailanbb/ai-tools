# Claude API 免费额度两周实测

上周有人问我能不能不花钱用 Claude Code。我整理了两周记录，写成这篇。

## 免费额度怎么来

#### ① 每月免费 token 上限

新账户每月有约 5 美元的抵扣额度。折算大概 300 万输入 token。

#### ② 实际消耗速度

跑一次代码审查用掉约 1 万 token。日常对话更少，每条约 500 token。

#### ③ 超额之后怎么处理

API 返回 `429` 状态码。等下月重置，或者直接充值。

## 本地方案对比

#### ① Ollama 本地模型

支持 Llama 3 和 Mistral 系列。隐私有保障，不走外网请求。代码补全质量比 Claude 差约 40%。

#### ② LM Studio

提供图形界面，对新手友好。响应速度约 3 秒每 token。写代码还行，解释代码经常答非所问。

## 写在最后

免费额度够轻度用户撑一个月。本地方案适合对隐私要求高的场景。两个都用也不矛盾。
