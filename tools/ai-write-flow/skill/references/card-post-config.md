# card-post-config.md — 图卡文案模式

> Step 0 路由到 card_post 模式时加载。本文件约束图卡输出的字数、卡片数量、格式和事实规则。

---

## 适用场景

当用户触发以下任意意图时，进入 card_post 模式：

- 公众号贴图 / 图卡文章 / 图卡配文
- 小红书图卡
- 「可复制到图卡生成系统」的结构化文案
- 明确要求「1000 字以内」的内容产品
- 用户提及「图卡 / 贴图 / 卡片 / 小红书 / card」

---

## 核心目标

生成一套可直接复制到图卡生成系统的结构化文案。

**图卡模式不是长文的摘要，是面向图片阅读的高密度信息表达。**

---

## 字数限制

- 图卡内容排版（封面 + 内容卡正文）总字数控制在 **1000 字以内**
- 发布配文字数不计入 1000 字限制，建议 200-350 字
- 事实核查摘要、卡片数量决策也不计入 1000 字限制（属于过程文档）
- 优先压缩废话，不压缩关键事实

| 卡片类型 | 建议字数 |
|---------|---------|
| 封面图卡 | 30-80 字 |
| 内容图卡 | 80-140 字 |

---

## 卡片数量规则

- 总卡片数最多 **9 张**（含封面）
- **不固定卡片数量** — 由信息单元数量决定，禁止为凑数重复观点
- 禁止为减少卡片遗漏关键数据

推荐参考（非强制）：

| 内容复杂度 | 推荐卡片数 |
|-----------|-----------|
| 单一功能更新 | 1 封面 + 3 内容卡 |
| 常规产品发布 | 1 封面 + 4-5 内容卡 |
| 重大版本发布 | 1 封面 + 5-8 内容卡 |
| 多产品对比 | 1 封面 + 6-8 内容卡 |

---

## 输出结构

输出必须按以下固定顺序：

```
【事实核查摘要】

- 已核查来源：
  1. [来源标题] [URL]
  2. [来源标题] [URL]

- 可安全写入图卡的事实：
  - [事实] → [来源]

- 不建议写入或必须弱化的事实：
  - [事实] → [原因]

【卡片数量决策】

建议：1 张封面 + N 张内容卡
原因：
- 信息单元 1：[说明]
- 信息单元 2：[说明]

未做成独立卡片的信息：
- [信息]：原因是与某卡片主题重复 / 证据不足 / 信息权重不够

【发布配文】

[适合公众号/小红书正文区的短文案，200-350 字，可包含话题标签]

【图卡内容排版】

【封面图卡】

| 主标题
| 副标题
|
| [主视觉：一句话描述视觉方向]
|
| 底部标签：#关键词1 #关键词2 #关键词3

【Card 01】

| 01 卡片标题
|
| 🟢 要点一
|
| 🟢 要点二
|
| 🟢 要点三

【Card 02】

| 02 卡片标题
|
| 🟢 要点一
|
| 🟢 要点二
|
| 🟢 要点三
|
| 🟢 要点四
```

---

## 每张内容卡规则

- 只有**一个**核心主题，标题必须具体（不写空泛口号）
- 要点数量 **3-4 条**（严格限制）
- 每条要点不超过 **24 个汉字**
- 优先使用数字、版本号、功能名、场景词
- 禁止「很强、效率提升、全面升级」等空泛表达
- 每张卡片之间不得重复同一个观点

## 封面图卡规则

封面必须包含：

- 产品名或主题名
- 一个**强钩子**标题（具体 + 利益点）
- 一个视觉方向描述
- 可选版本号或发布代号
- 不超过 2 行主文案

示例：

```
【封面图卡】

| Hermes Agent 重磅更新！
| 告别命令行，全场景桌面端来了。
|
| [主视觉：v0.16 SURFACE RELEASE]
|
| 底部标签：#AI #Agent #HermesAgent #生产力
```

---

## 事实核查规则

图卡模式的事实核查比长文更严格，因为短文案更容易把事实写绝对。

所有以下信息必须有**官方来源**支撑才能写入卡片：

- 版本号、发布时间
- 性能提升数据
- 新功能名称
- 支持平台
- 安装命令
- 模型名称
- 官方集成
- 架构变化
- 价格或限制

**`unverified` 事实禁止进入发布内容（【发布配文】和【图卡内容排版】）。** 已排除的事实必须列入【事实核查摘要】的「不建议写入」栏。

详细事实核查规则见 `references/fact-policy.md`。

---

## 禁止行为

- 禁止说「100% 准确」，只能说「当前已由官方来源核验」
- 禁止把未核验信息写成确定事实
- 禁止为爆款牺牲事实准确性
- 禁止为凑卡片数重复观点
- 禁止输出长篇解释塞进图卡内容排版
